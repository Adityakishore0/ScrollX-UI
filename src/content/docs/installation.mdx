## Requirements

Before you begin, make sure you have:

- Node.js 16 or higher
- npm, yarn, or pnpm installed

## Basic Installation

```sh
npm install @mylib/core
```

## Configuration

Create a configuration file in your project root:

```js
// mylib.config.js
module.exports = {
  theme: "light",
  components: ["button", "card", "input"],
};
```
