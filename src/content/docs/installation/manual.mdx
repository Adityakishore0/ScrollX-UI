<Steps>

<Step>Add Tailwind CSS</Step>

Components are styled using `Tailwind CSS`. You need to install Tailwind CSS in your project.

[Follow the Tailwind CSS installation instructions to get started.](https://tailwindcss.com/docs/installation/framework-guides/nextjs)

<Step>Add Dependencies</Step>

Integrate the following dependencies into your project:

<DepsOptions name='class-variance-authority motion clsx tailwind-merge lucide-react tw-animate-css' />

<Step>Set up path aliases</Step>

Set up the path aliases in your `tsconfig.json` file.

<code>tsconfig.json</code>
```tsx
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./*"]
    }
  }
}
```

<Step>Add util file</Step>

`lib/utils.ts`

```tsx
import { ClassValue, clsx } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
````

<Step>Create a components.json file</Step>

Create a <code>components.json</code> in the root of your project.

components.json

```tsx
{
 "$schema": "https://ui.shadcn.com/schema.json",
 "style": "new-york",
 "rsc": false,
 "tsx": true,
 "tailwind": {
   "config": "",
   "css": "src/styles/globals.css",
   "baseColor": "neutral",
   "cssVariables": true,
   "prefix": ""
 },
 "aliases": {
   "components": "@/components",
   "utils": "@/lib/utils",
   "ui": "@/components/ui",
   "lib": "@/lib",
   "hooks": "@/hooks"
 },
 "iconLibrary": "lucide"
}
```
