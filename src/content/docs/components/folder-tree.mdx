## Usage

```tsx
import { FolderTree } from "@/components/ui/folder-tree";
```

### Basic Usage

```tsx
  <FolderTree.Root
        defaultExpanded={["root", "folder1"]}
        defaultSelected="file1"
        className="w-full"
      >
        <FolderTree.Item id="root" label="Root Folder">
          <FolderTree.Content>
            <FolderTree.Item id="folder1" label="Documents">
              <FolderTree.Content>
                <FolderTree.Item id="file1" label="File 1.txt" />
                <FolderTree.Item id="file2" label="File 2.txt" />
                <FolderTree.Item id="file3" label="Image.png" />
              </FolderTree.Content>
            </FolderTree.Item>
          </FolderTree.Content>
        </FolderTree.Item>
      </FolderTree.Root>
```

### FolderTree.Root

The root component that provides context and manages global state.

### FolderTree.Item

Represents individual folder or file nodes. Requires `id` and `label` props.

### FolderTree.Content

Wraps child items and provides smooth expand/collapse animations. Use this to contain nested items within a folder.

### FolderTree.Trigger

Optional component for custom expand/collapse behavior. The expand/collapse functionality is built into the Item component by default, so this is only needed for advanced customization.
