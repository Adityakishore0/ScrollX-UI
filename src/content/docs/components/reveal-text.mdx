## API Reference

### Reveal Text

a clean text reveal component providing directional animation and stagger options.

## Props


| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `children` | `ReactNode` | `-` | Text or elements to animate. |
| `className` | `string` | `-` | Classes applied to the visible text. |
| `boxClassName` | `string` | `-` | Classes applied to the reveal animation layer. |
| `delay` | `number` | `0` | Animation delay before the reveal starts. |
| `duration` | `number` | `0.8` | Duration of the reveal animation. |
| `direction` | `"up" \| "down" \| "left" \| "right"` | `"down"` | Direction from which the reveal layer moves. |
| `mode` | `"manual" \| "auto"` | `"manual"` | Animate full text or auto-split into words. |
| `stagger` | `number` | `0.1` | Delay between word animations in auto mode. |
| `once` | `boolean` | `true` | Run animation once or every time it enters view. |
