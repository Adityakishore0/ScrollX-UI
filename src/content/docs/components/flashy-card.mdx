## Usage

```tsx

```

```tsx

    className="w-full max-w-sm"
  defaultSrc="https://images.pexels.com/photos/1933316/pexels-photo-1933316.jpeg"
  activeSrc="https://www.pexels.com/download/video/1390942/"
  activeType="video"
/>
```
