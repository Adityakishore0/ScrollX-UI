## API Reference

### StaggerChars

The `StaggerChars` component renders an animated text component with staggered character effects.

## Props

| Prop                  | Type                            | Default              | Description                                                |
| --------------------- | ------------------------------- | -------------------- | ---------------------------------------------------------- |
| `text`                | `string`                        | `(required)`         | The base text to display                                   |
| `hoverText`           | `string`                        | `text`               | Text to show on hover (defaults to `text` if not provided) |
| `delay`               | `number`                        | `0.05`               | Delay between each character's animation in seconds        |
| `duration`            | `number`                        | `1`                  | Duration of the animation in seconds                       |
| `className`           | `string`                        | —                    | Additional classes for the container                       |
| `hoverClassName`      | `string`                        | —                    | Classes for the hover text                                 |
| `direction`           | `'up' \| 'down' \| 'alternate'` | `'alternate'`        | Animation direction                                        |
| `easing`              | `number[]`                      | `[0.22, 1, 0.36, 1]` | Custom easing function for the animation                   |
| `disabled`            | `boolean`                       | `false`              | Disable the hover effect                                   |
| `onAnimationStart`    | `() => void`                    | —                    | Callback when animation starts                             |
| `onAnimationComplete` | `() => void`                    | —                    | Callback when animation completes                          |
