## API Reference

### Draggable Avatar

It displays a circular, draggable avatar with customizable size and style.

## Props


| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `image` | `string` | `-` | URL of the avatar image displayed inside the draggable circle. |
| `borderColor` | `string` | `"#60A5FA"` | Color of the avatarâ€™s circular border and the bottom marker line. |
| `range` | `number` | `300` | Maximum drag distance (in pixels) allowed from the initial position. |
| `size` | `number` | `100` | Diameter of the circular avatar in pixels. |
