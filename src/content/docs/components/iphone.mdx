## API Reference

### Iphone

IPhone mockup with screen, header, and custom children content.

## Props


| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `imgUrl` | `string` | `_` | URL of the image displayed on the iPhone screen. |
| `clip` | `string` | `_` | URL of a video displayed on the iPhone screen. |
| `showHeader` | `boolean` | `false` | Whether to show the iPhone status bar (time, battery, wifi). |
| `headerClassName` | `string` | `_` | Optional CSS class to customize the status bar styling. |
| `className` | `string` | `_` | CSS class for the outer iPhone container. |
| `style` | `React.CSSProperties` | `_` | Inline styles for the iPhone container. |
| `children` | `React.ReactNode` | `_` | Content displayed inside the iPhone screen above image/video. |
