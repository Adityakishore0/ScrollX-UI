## Usage

```tsx

```

```tsx
<CodeBlock
  language="typescript"
  filename="main.tsx"
  tabs={[
    { name: 'api.ts', code: "fetchData()" },
    { name: 'styles.css', code: ".card { padding: 1rem }", language: 'css' }
  ]}
  breadcrumb={['src', 'components']}
/>
```

## API Reference

### CodeBlock

Syntax-highlighted code block with tabs, stats, and file navigation.

#### Props  
**`language`** `string` (required)  
**`filename`** `string` (required)  
**`code`** `string` (required without tabs)  
**`tabs`** `Array<{name: string; code: string; language?: string}>` (required without code)  
**`highlightLines`** `number[]` (default `[]`)  
**`breadcrumb`** `string[]` (default `[]`)  
**`showStats`** `boolean` (default `true`)  
**`theme`** `'dark' | 'light'` (default `'dark'`)
