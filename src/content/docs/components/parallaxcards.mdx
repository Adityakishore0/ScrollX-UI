## API Reference

### `ParallaxCards`

A vertical sticky-scroll component that displays stacked cards with parallax behavior.

## Props


| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `cards` | `{ lightBg: string; darkBg: string; content: React.ReactNode; }[]` | `[]` | Array of cards with light/dark background and content. |
| `height` | `number \| string` | `"100vh"` | Height of the scroll container. Each card is sticky (70vh tall). |
