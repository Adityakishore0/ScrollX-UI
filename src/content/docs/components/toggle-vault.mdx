## Usage

```tsx

  ToggleVault,

  ToggleVaultContent,
```

## API Reference

### Toggle Vault

`ToggleVault` is a flexible, animated side menu component built with Framer Motion and Tailwind CSS.

### ToggleVaultTrigger

The button that toggles the panel open/close. Accepts either static text/icons or a render prop that receives `{ open }`.

Props:

- `children`: `ReactNode | ({ open: boolean }) => ReactNode`
- Content of the toggle button. Can be a string, JSX, or function render prop.
- `className`: `string` (optional) — Additional Tailwind classes for styling.

### ToggleVaultContent

The panel that animates in/out. You can place links, components, or any custom layout inside.

Props:

- `children`: `ReactNode` — The panel’s content.
- className: string (optional) — Custom styling, including width and height.

### ToggleVaultClose

Optional. Typically not needed if links auto-close the panel.

Props:

- `children`: `ReactNode` — Content of the close button (text, icon, etc).
- `className`: `string` (optional) — Custom styling for the button.
