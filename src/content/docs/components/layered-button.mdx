## API Reference

### Layered Button

A layered button with a radial expand effect and smooth sliding hover label animation.

## Props


| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `variant` | `string` | `default` | Button style. |
| `size` | `string` | `default` | Button size. |
| `borderWidth` | `number` | `3` | Inner border width. |
| `asChild` | `boolean` | — | Use Radix Slot. |
| `children` | `node` | — | Content inside button. |
| `className` | `string` | — | Custom classes. |
| `...props` | `button props` | — | All native button attributes. |
