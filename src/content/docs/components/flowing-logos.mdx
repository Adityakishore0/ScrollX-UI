## API Reference

### Flowing Logos

Flowing logo showcase with smooth animations, hover effects, and modern UI.

## Props

| Prop            | Type          | Default     | Description                                                |
| --------------- | ------------- | ----------- | ---------------------------------------------------------- |
| `data`          | `Array<Logo>` | `required`  | An array of logo objects to display.                       |
| `name`          | `string`      | `required`  | Name of the logo or brand. Used inside each logo object.   |
| `image`         | `string`      | `required`  | URL of the logo image. Used inside each logo object.       |
| `className`     | `string`      | `undefined` | Optional additional CSS classes applied to the container.  |
| `cardClassName` | `string`      | `undefined` | Optional additional CSS classes applied to each logo card. |
