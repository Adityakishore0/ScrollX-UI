## API Reference

### Flowing Logos

Flowing logo showcase with smooth animations, hover effects, and modern UI.

## Props

| Prop            | Type                               | Default      | Description                                                     |
| --------------- | ---------------------------------- | ------------ | --------------------------------------------------------------- |
| `data`          | `Array<{ name: string; image: string }>` | `required`   | Array of logo objects to render in the marquee.                |
| `variant`       | `'square' \| 'wide' \| 'auto'`     | `'square'`   | Controls the logo card layout and sizing behavior.             |
| `className`     | `string`                           | `-`  | Additional classes applied to the outer container.             |
| `cardClassName` | `string`                           | `-`  | Additional classes applied to each individual logo card.       |
