## API Reference

### Kbd

A component for displaying keyboard shortcuts and key combinations.

## Props


| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `children` | `ReactNode` | `required` | The content inside the Kbd component, usually the key label or elements. |
| `className` | `string` | `undefined` | Optional additional CSS classes to customize styling. |
| `...props` | `ComponentProps<'span'>` | `undefined` | All other native span element props are supported. |
