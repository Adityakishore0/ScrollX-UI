- [<u>Striped Grid</u>](/docs/components/striped-grid): Animated diagonal stripes over an optional base grid.  
- [<u>Drift Card</u>](/docs/components/drift-card): Interactive card with a subtle drift hover highlight.  
- [<u>Pixel Background</u>](/docs/components/pixel-background): Flexible pixel field with smooth atmospheric motion.  
- [<u>Electric Text</u>](/docs/components/electric-text): Neon-style text with electric glow animation.  
- [<u>Radio Group</u>](/docs/components/radio-group): Clean single-select radio button group.  
- [<u>Expandable Cards</u>](/docs/components/expandable-cards): Cards that expand on hover for focus.  
- [<u>Decorated Text</u>](/docs/components/decorated-text): Text framed with minimal decorative corners.  
- [<u>Card Tilt</u>](/docs/components/card-tilt): Smooth 3D tilt effect reacting to cursor movement.  
- [<u>Reveal Text</u>](/docs/components/reveal-text): Text reveal animation with directional control.  
- [<u>Motion Navbar</u>](/docs/components/motion-navbar): Animated navbar with dropdowns and mobile menu.  
- [<u>Motion Grid</u>](/docs/components/motion-grid): Diagonal grid with continuous glowing motion.  
- [<u>Shiny Button</u>](/docs/components/shiny-button): Button with a cursor-tracking spotlight glow.  
- [<u>Stagger Button</u>](/docs/components/stagger-button): Button with animated staggered text on hover.
